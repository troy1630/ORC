# ORC Branding Direction

## Brand Concept

ORC should feel like a command chamber for intelligent operational guardians. The mascot is a mystical orc: not comic or brutish, but strategic, arcane, and vigilant.

## Visual Guidance

- Tone: arcane operations room, not fantasy parody
- Palette: obsidian, tarnished brass, ember, moss, parchment
- UI metaphor: agent messages travel as visible paths between shrines, towers, or stations
- Motion: meaningful movement that shows workflows, approvals, and blocked actions

## Logo Direction

- Orc silhouette with rune-like signal lines or constellation links
- Optional staff, crystal, or command sigil to communicate coordination
- Avoid generic gaming logos that imply violence or aggression

## UI Direction

- Start with a readable control-room dashboard
- Layer the character-based view over real system state
- Make approvals, incidents, and evidence the visual center of gravity
